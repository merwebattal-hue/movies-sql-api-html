# Movie App 🎬

A command-line movie management application built with **Python**, **SQLite**, and the **OMDb API**.
Users can manage their personal movie collection and generate a simple movie website.

## Features

* Add movies via OMDb API
* Store movies in SQLite database
* Multi-user profiles
* Movie statistics
* Random movie suggestion
* Generate a personal movie website
* Add personal notes to movies

## Technologies

* Python
* SQLAlchemy
* SQLite
* OMDb API
* HTML / CSS

## How to Run

Clone the repository:

```
git clone https://github.com/merwebattal-hue/movies-sql-api-html.git
```

Enter the folder:

```
cd movies-sql-api-html
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python movies.py
```

Generate a movie website directly from your movie collection.
